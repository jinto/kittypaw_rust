왜 KittyPaw인가요?

n8n, Zapier 같은 자동화 도구의 데스크톱 네이티브 대안. AI가 코드를 쓰고, 샌드박스에서 안전하게 실행합니다.

🎯

코드 없이 자동화

스킬 갤러리에서 원하는 자동화를 설치하고, 텔레그램 토큰만 넣으면 바로 동작합니다.

💬

AI 채팅

데스크톱 앱에서 바로 Claude와 대화. 질문하고, 분석하고, 자동화를 지시하세요.

🔒

안전한 실행

모든 스킬은 QuickJS 샌드박스 + OS 커널 격리 (Seatbelt/Landlock) 안에서 실행됩니다.

🔗

스킬 체이닝

여러 스킬을 순차 파이프라인으로 연결. 데이터 수집 → 요약 → 발송을 한 번에.

🌐

웹 검색

스킬이 웹을 검색하고, 페이지를 읽고, AI로 요약합니다. 팩트체크도 자동으로.

🦀

순수 Rust

JS, CSS, npm 없이 Rust만으로 구축. Dioxus 데스크톱 GUI. 단일 바이너리 배포.
